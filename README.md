# 时间管理大师

## 使用方法

### 方法一：直接打开（推荐，无需安装任何东西）
直接用浏览器打开 `public/index.html` 即可使用。
所有数据保存在浏览器 localStorage 中。

### 方法二：使用 Node.js 服务端（可选）
```bash
npm install
npm start
```
然后访问 http://localhost:3000

## 功能

- **邮箱注册/登录**：使用邮箱和密码注册账号
- **日历手账**：按月查看日历，点击日期查看/编写手账，支持 5 个分类
- **待办事项**：管理未完成的事项，支持 5 个分类、3 级优先级、截止日期
- **分类系统**：健身💪、影视🎬、学习📚、工作💼、日常🌟

## 设计风格

高端冷色风：深空蓝背景 + 冰蓝点缀 + 玻璃态卡片 + 微光粒子效果
